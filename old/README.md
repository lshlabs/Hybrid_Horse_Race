# Old Files Directory

이 디렉터리는 프로젝트 정리 과정에서 사용되지 않는 파일들을 보관하기 위해 생성되었습니다.

## 이동된 파일/디렉터리 목록

### 1. frontend_public_horses/
- **이전 위치**: `frontend/public/assets/images/horses/`
- **이동 이유**: 중복된 말 이미지 파일 (8개). 실제로는 `frontend/src/assets/images/horses/`에서 사용 중
- **이동일**: 2026-01-23

### 2. frontend_public_track/
- **이전 위치**: `frontend/public/assets/images/track/`
- **이동 이유**: 사용되지 않는 트랙 이미지 파일들 (100개 이상). 실제로는 `frontend/src/assets/images/map/map2.png`만 사용됨
- **이동일**: 2026-01-23

### 3. frontend_src_assets_map/map1.png
- **이전 위치**: `frontend/src/assets/images/map/map1.png`
- **이동 이유**: 사용되지 않는 맵 이미지. 실제로는 `map2.png`만 사용됨
- **이동일**: 2026-01-23

### 4. img_assets/
- **이전 위치**: 프로젝트 루트의 `img_assets/`
- **이동 이유**: 프로젝트에서 사용되지 않는 이미지 에셋 디렉터리. frontend 프로젝트 구조와 맞지 않음
- **이동일**: 2026-01-23

### 5. frontend_test-phaser.html
- **이전 위치**: `frontend/test-phaser.html`
- **이동 이유**: 사용되지 않는 개발용 테스트 HTML 파일. 실제 구현은 `PhaserGame.tsx`에 통합됨
- **이동일**: 2026-01-23
